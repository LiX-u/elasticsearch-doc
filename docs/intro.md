# Elasticsearch 翻译说明

基于官方版本 7.11 文档开始，当前版本为 8.12。由于官方版本更新比较快，文档虽然是基于 7.11 开始翻译，，实际翻译过程中，后续的文档都是翻译时当时的最新版本。

计划在全部翻译完成后，再基于完成时的最新版本，对有差异的老版本文档进行调整。
