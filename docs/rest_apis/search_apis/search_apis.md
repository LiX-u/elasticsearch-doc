# 搜索 API

搜索 API 用于搜索和聚合存储在 Elasticsearch 索引和数据流中的数据。相关关概述和相关教程，参阅[搜索你的数据](/search_your_data)。

大多数搜索 API 支持 [多目标语法](/rest_apis/api_convention/multi_target_syntax)，除了[解释 API](/rest_apis/search_apis/explain)。
